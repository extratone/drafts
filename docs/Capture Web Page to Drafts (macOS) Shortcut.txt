## Feed Safari Reader content from the clipboard to capture a web page in Markdown.

![CopyingSafariReaderContents](https://user-images.githubusercontent.com/43663476/153740311-a7076601-bd68-48ec-a002-cdabe6d681c4.png)

![CaptureWebPagetoDraftsmacOS](https://user-images.githubusercontent.com/43663476/153740270-09106caa-05eb-49a2-876f-d6e7091a0a2a.png)

This shortcut circumvents the Safari Reader macOS Shortcuts problem by **expecting you've copied Safari Reader Contents (⇧⌘R) to the clipboard**.
