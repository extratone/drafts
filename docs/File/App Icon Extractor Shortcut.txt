## Extract file/folder/app icons as .pngs.

![File-App Icon Extractor](https://user-images.githubusercontent.com/43663476/152102068-248d15bb-b0e7-4308-94dd-cd479c37af76.png)

### Requirements
[Actions for macOS](https://apps.apple.com/us/app/actions/id1586435171)

This shortcut uses [Actions for macOS](https://apps.apple.com/us/app/actions/id1586435171)' `Get File Icon` action to extract the icon images of apps, folders, and files as .pngs.
