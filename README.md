*Definitely read [Tim Nahumck‚Äôs work on this feature](https://nahumck.me/drafts-29-autocomplete/) if you‚Äôre curious about it.*

#### Install
The hyperlinks on the expansion triggers will take you a corresponding plain text file containing the installation URL (which is, of course, the entire contents of each, encoded in a string.) Copy the full text of the URL, paste it in your web browser, and tap/click ‚ÄúGo‚Äù (or your equivalent)/key `RETURN` or `ENTER`. Regardless of platform, you should receive a prompt asking if you‚Äôd like to open it in Drafts.
