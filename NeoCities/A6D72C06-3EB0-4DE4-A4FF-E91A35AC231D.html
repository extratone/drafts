nick PopClip Developer ￼ 22 Oct 24 MacOS 12 Monterey is due to be released tomorrow (Monday 25th October) so here is a quick update on how PopClip fits in.

PopClip itself is fully compatible with Monterey and doesn’t have any specific issues (that I know of). As long as you have the latest version, PopClip 2021.11, it should work fine. There are often glitches on the first boot after installing a new OS. If you have any trouble with PopClip after installing Monterey, please first try rebooting the Mac one more time.

However you will need to download updates to some PopClip extensions, and some extensions do not yet work on Monterey.
